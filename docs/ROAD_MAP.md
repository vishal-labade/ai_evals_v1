# Roadmap

## V1 (v1_final) — complete
- Stable prompt suite + deterministic expansions
- Deterministic scoring + optional canonical numeric scoring for exact tasks
- NIC metrics: global, conditional, and explicit k/n
- Wilson confidence intervals
- Stable leaderboard artifacts

- Multi-turn test cases with state
- Instruction hierarchy tests (system vs user)
- Consistency over turns
- Robustness to paraphrase and distractors (behavioral)
- Safety/refusal behavior tests (policy-aligned)
